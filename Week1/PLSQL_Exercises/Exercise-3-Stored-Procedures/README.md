# Exercise 3 - Stored Procedures

## Objective

The objective of this exercise is to understand and implement PL/SQL stored procedures for common banking operations.

---

## Scenario 1

Process monthly interest for all savings accounts by applying 1% interest to the account balance.

---

## Scenario 2

Update employee salaries by adding a bonus percentage for a specified department.

---

## Scenario 3

Transfer funds between two accounts after checking whether the source account has sufficient balance.

---

## Concepts Used

* CREATE PROCEDURE
* Parameters (IN)
* UPDATE statement
* IF-ELSE
* SELECT INTO
* COMMIT
* DBMS_OUTPUT.PUT_LINE

---

## Procedures Created

1. ProcessMonthlyInterest
2. UpdateEmployeeBonus
3. TransferFunds

---

## Conclusion

This exercise demonstrates how PL/SQL stored procedures automate banking operations while improving code reusability and maintainability.
