# Exercise 4 - Arrange-Act-Assert (AAA) Pattern, Setup and Teardown in JUnit

## Objective

The objective of this exercise is to understand the Arrange-Act-Assert (AAA) testing pattern and use JUnit setup and teardown methods.

## Technologies Used

* Java
* Maven
* JUnit 4.13.2

## Files Included

* Calculator.java
* CalculatorTest.java
* pom.xml

## Concepts Demonstrated

* Arrange-Act-Assert (AAA) Pattern
* @Before Annotation
* @After Annotation
* assertEquals()

## Expected Result

The test executes successfully and verifies that the Calculator addition method returns the expected value.

Tests Passed: 1/1

## Conclusion

The AAA pattern improves the readability and organization of test cases, while the @Before and @After methods help initialize and clean up resources before and after each test.
