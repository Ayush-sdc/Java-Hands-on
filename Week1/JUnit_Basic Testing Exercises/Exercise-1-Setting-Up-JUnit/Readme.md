# Exercise 1 - Setting Up JUnit

## Objective

The objective of this exercise is to set up JUnit in a Java project and write a simple unit test.

---

## Steps Performed

1. Created a Maven Java project.
2. Added the JUnit 4.13.2 dependency in the `pom.xml` file.
3. Created a simple `Calculator` class.
4. Created a JUnit test class named `CalculatorTest`.
5. Verified the functionality using `assertEquals()`.

---

## Files Included

* Calculator.java
* CalculatorTest.java
* pom.xml

---

## Concepts Used

* Maven
* JUnit 4
* @Test Annotation
* assertEquals()

---

## Expected Output

The unit test executes successfully and verifies that the addition method returns the expected result.

---

## Conclusion

JUnit provides a simple framework for writing and executing unit tests. It helps verify the correctness of Java code automatically.
